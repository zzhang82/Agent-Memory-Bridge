# v0.15.0 - Reviewed Memory Revision Gates

Agent Memory Bridge 0.15.0 is a narrow governance release over the
learning-candidate lane. It keeps the public MCP surface stable while making
reviewed memory revision and forgetting gates measurable.

`0.15.0 = review receipts and deterministic memory-evolution proof without
autonomous mutation.`

## What Changed

- Learning review records now include deterministic review-receipt hashes.
- Review receipts explicitly carry `writeback_boundary:review_receipt_only`.
- Review receipts explicitly state `durable_mutation_performed_by_review: false`.
- A new memory-evolution fixture benchmark checks reviewed supersession,
  deletion/tombstone audit, quarantine, principal-scope warnings, bitemporal
  validity, and hidden review lanes.
- The release contract now checks the memory-evolution snapshot facts alongside
  the existing retrieval, calibration, procedure, signal contention, and
  adversarial governance snapshots.

## Honest Boundary

This release does **not** claim:

- autonomous self-evolving memory
- automatic durable promotion from raw transcripts
- ACL enforcement or enterprise authorization
- GDPR/privacy compliance
- certified poisoning resistance
- a graph database or graph-memory platform
- a scheduler, runtime, watcher product, or UI

It adds local, repeatable proof that AMB can keep reviewed revision artifacts
auditable while preventing candidates, deleted records, quarantined claims, and
obsolete records from becoming normal durable authority by accident.
