# v0.19.0 - Proof Breadth Before Product Surface

Agent Memory Bridge 0.19.0 is a proof-breadth release over the v0.18 Task Brief
and first-run adoption work.

`0.19.0 = widen proof before widening product surface.`

## What Changed

- Added a fixed 12-case v0.19 adoption-proof pack:
  - 4 retrieval cases for durable decisions, gotchas, procedures, and concepts
  - 4 Task Brief cases for `Used`, `Ignored`, `Needs Review`, and active signals
  - 4 first-run guidance cases for generic stdio, Codex, Claude Code, and Cursor
- Added an executable runner:
  - `python scripts/run_v019_adoption_proof.py`
- Added a generated snapshot:
  - `benchmark/latest-v0.19-adoption-proof-report.json`
- Added release-contract facts for the v0.19 denominator and pass rates.
- Kept the public MCP surface unchanged at 10 tools.

## Validation

- `pytest`: `323 passed`
- release contract: passed
- public MCP surface: unchanged at `10` tools
- v0.19 adoption-proof pack: `12 / 12` cases passed

## Boundaries

The v0.19 adoption-proof report is synthetic fixture proof only. It is not a
claim of clean-room external adoption.

This release is still not:

- a new MCP tool
- an AMH runtime dependency
- a client config writer
- a plugin package for every client
- a scheduler, watcher, worker, or runtime loop
- an automatic promotion, deletion, merge, approval, or writeback path
