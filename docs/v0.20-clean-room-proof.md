# v0.20 Clean-Room Adoption Proof

This is the bounded proof lane for the `0.20.0` release.

`0.20 = clean-room adoption before product expansion.`

## Proof Contract

The runner must prove a local fresh-start path without changing public runtime
behavior:

- create an isolated temporary bridge home and store
- launch the real local stdio entrypoint with `python -m agent_mem_bridge`
- perform one explicit demo `store -> recall` memory round trip over MCP stdio
- render first-run output from the temp store
- render Task Brief output from the temp store
- write a machine-readable JSON report and a short Markdown transcript
- keep client config output manual-copy only
- leave the configured client config path absent
- keep AMH absent from the required path
- keep the public MCP surface at 10 tools

The proof writes only explicit demo memory inside the temporary store. First-run
and Task Brief rendering must not mutate the temp store.

## Fixed Case Denominator

The proof runner has exactly `6` cases:

1. local entrypoint/import sanity
2. stdio MCP public tool-surface check
3. tokened `store -> recall` over MCP stdio
4. `first-run` CLI output
5. `task-brief` CLI output
6. isolated write-scope guard

This denominator is intentionally small. It proves a fresh local path through
the product without turning the release into plugin packaging, client
certification, runtime scheduling, or broader adoption claims.

## Run

```bash
python scripts/run_v020_clean_room_proof.py \
  --report-path benchmark/latest-v0.20-clean-room-proof-report.json \
  --transcript-path benchmark/latest-v0.20-clean-room-proof-transcript.md
```

To preserve the temporary runtime for inspection, pass an absent or empty
runtime directory:

```bash
python scripts/run_v020_clean_room_proof.py \
  --runtime-dir .tmp/v020-clean-room \
  --report-path .tmp/v020-clean-room/report.json \
  --transcript-path .tmp/v020-clean-room/transcript.md
```

## Validation

```bash
python -m pytest tests/test_v020_clean_room_proof.py tests/test_cli_verify.py tests/test_cli_first_run.py tests/test_task_brief.py tests/test_stdio_integration.py
python scripts/check_public_surface.py
python scripts/check_release_contract.py
```

## Non-Claims

This proof is local reproducible evidence. It is not:

- external vendor certification
- a client config writer
- a plugin package
- a new MCP tool
- an AMH dependency
- a watcher, scheduler, daemon, or runtime loop
- automatic promotion, deletion, approval, merge, or durable writeback outside
  the temp proof store
