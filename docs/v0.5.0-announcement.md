# v0.5.0 Announcement

## English

Agent Memory Bridge v0.5.0 is out.

Two-channel MCP memory for coding agents:
- durable knowledge in `memory`
- coordination events in `signal`

This release makes two things much stronger:

1. Retrieval is now measurable and tunable
- canonical benchmark
- `precision@k`
- `expected_top1_accuracy`
- current canonical fixture: `memory_expected_top1_accuracy = 1.0`

2. Signal lifecycle is fuller
- `claim -> extend -> ack / expire / reclaim`
- `extend_signal_lease` is now part of the MCP surface
- hard expiry stays a real upper bound

Also included:
- deterministic proof for signal correctness, duplicate suppression, and recall timing
- `68 passed`
- short terminal demo GIF in the repo

## 简体中文

Agent Memory Bridge v0.5.0 发布了。

这是一个面向编码代理的双通道 MCP 记忆层：
- `memory` 负责持久知识
- `signal` 负责协调事件

这次最重要的两件事：

1. retrieval 现在可度量、可调优
- 有 canonical benchmark
- 会看 `precision@k`
- 也会看 `expected_top1_accuracy`
- 当前 canonical fixture 上：`memory_expected_top1_accuracy = 1.0`

2. signal lifecycle 更完整了
- `claim -> extend -> ack / expire / reclaim`
- `extend_signal_lease` 已进入公开 MCP surface
- hard expiry 仍然是严格上限

另外这版还有：
- 针对 signal correctness、duplicate suppression、recall timing 的 deterministic proof
- `68 passed`
- repo 里已经带了一个很短的终端 demo GIF
